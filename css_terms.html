CSS

color: value
changes font color.
background-color: value
changes background color with hex, rgb, hsl.
font-family: list
comma separated list that goes from most to least specific font.
font-size: value
changes the font size.
text-align: value
changes text alignment to center, left, right, or justify.
text-decoration: value
changes decoration of text, underline, or none.
display: value
changes the way the content is displayed like inline or block.
border-style: value
changes the border type like dotted, dashed, solid, etc.
padding: value
changes the padding space of the element.
box-sizing: value
change how things get displayed content or border-style.
float: value
changes the position of the element on left or right margin.
clear: value
pushes the element down until there are no floats on the specified side(s).
outline: width type color
creates an outline around the specified element.
line-height
changes the height of the lines.
letter-spacing: value
changes the spacing between letters.

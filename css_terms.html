CSS

color: value;
changes font color.
background-color: value
changes background color with hex, rgb, hsl.
text-align: value
changes text alignment to center, left, right, or justify.
display: value
changes the way the content is displayed like inline or block.
border-style: value
changes the border type like dotted, dashed, solid, etc.
padding: value
changes the padding space of the element.
box-sizing: value
change how things get displayed content or border-style.
float: value
changes the position of the element on left or right margin.
clear: value
pushes the element down until there are no floats on the specified side(s).
